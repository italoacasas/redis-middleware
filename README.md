Express Middleware to handle Redis connections

## Installation
`npm install --save redis-middleware`

## Example
```javascript
const app = require('express')();
const redisMiddleware = require('redis-middleware');

app.use(redisMiddleware())

app.post('/', (req, res) => {
    req.redis.set(key, value);  
}
```
